Ce programme est basé sur le script "spot dl".
Pour pourvoir le lancé il faut avoir préalablement :
	- Téléchargé python : https://www.python.org/downloads/
	- Téléchargé le programme python "spot dl" 
		- Il suffit de faire la commande suivante dans le cmd : pip3 install spotdl
		- Pour plus d'informations, voici la page GitHub du script : https://github.com/ritiek/spotify-downloader

Après avoir fait ces pré-requis :
	- Il vous suffit de configurer le fichier conf.txt
		- La partie connexion correspond aux identifiants de la base de données
		- La partie config contient deux dossier à indiquer :
			- Le dossier "musiques" du projet
			- Le dossier "musique" de l'utilisateur du pc

Le fichier config est prérempli pour example.
